My Sample Work

Below are three projects that I am most proud of from this semester. These assignments helped me understand both the theory behind computer science and how to apply programming concepts in code.

Project 1: Computer Architecture Slideshow

This project was a multi-part slideshow assignment focused on computer architecture. It was divided into three sections: microprocessors (Part A), digital and analog devices (Part B), and modern memory and mass storage devices (Part C). The purpose of this assignment was to show my understanding of how computers are structured and how different hardware components work together.

Project 2: Integer Calculation Program

This programming assignment uses Python to ask the user for three integer numbers. The program multiplies the three numbers together, finds their total, and then subtracts the total from the product. The final result is printed in a sentence that includes the numbers entered by the user and the calculated answer. This assignment helped me practice working with user input, variables, and mathematical operations.

Project 3: List, Loop, and Average Program

In this Python assignment, the program uses a loop to ask the user for five numbers and stores them in a list. A for loop is then used to calculate the total of the numbers, followed by calculating the average. The program prints both the total and the average in a clear sentence format. This project helped me better understand lists, loops, and how to work with numerical data.
